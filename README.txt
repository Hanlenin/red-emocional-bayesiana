
# Red Emocional Bayesiana Interactiva

Esta app en Streamlit te permite construir tu propia red emocional basada en eventos de tu vida y reacciones automáticas, usando inferencia bayesiana.

## Cómo usar
1. Instala Streamlit si no lo tienes:
   pip install streamlit

2. Ejecuta la app:
   streamlit run app.py
